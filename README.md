## Файловая система
*Функции:*
1. Добавление данных
2. Удаление данных
3. Изменение данных
4. Вывод данных
5. Копирование - функция копирует выбранную строку из одного файла, переносит ее в конец другого файла

**ui.py** - отвечает за взаимодействие программы с пользователем

**main.py** - главный файл, который вызывает всю цепочку

**db** - папка, в которой хранится 2 файла.
